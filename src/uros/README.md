This directory contains the micro-ROS agent and micro-ROS messages that are used
to interact with the CCB. This directy was automatically generated using the
`create-agent.sh` and `build-agent.sh` scripts of the 
[micro_ros_setup](https://github.com/micro-ROS/micro_ros_setup) project. 

If the agent needs to be updated in the future, follow the instructions of 
[micro_ros_setup](https://github.com/micro-ROS/micro_ros_setup) to create an agent 
and then replace this directory with the generated "uros" directory.
